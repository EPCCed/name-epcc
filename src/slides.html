Pattern, Policy, and Loop Body


/* Consider a loop: */

for (int index = 0; index < nIterations; index++) {
   result[index] = 2*index;
}

/* Pattern... */
for (...)

/* Execution Policy... */
(int index = 0; index < nIterations; index++)

/* Body... */
result[index] = 2*index;

A combination of pattern and policy drives execution of the body
